## Custom aliases to import Components & Assets etc.

Ready to use aliases for `import` paths :

| Alias         | Target Path                 |
| ------------- | --------------------------- |
| `$src`        | `./src`                     |
| `$assets`     | `./assets`                  |
| `$main`       | `./src/main`                |
| `$renderer`   | `./src/renderer`            |
| `$components` | `./src/renderer/components` |
| `$styles`     | `./src/renderer/styles`     |
